How to use this report

Backtest

REBAL_DATE: Date when the portfolio was formed. N_STOCKS: Number of stocks passing the screen. PORTFOLIO_RET_1Y: Equal-weight return of those stocks over the next year. BENCH_RET_1Y: Nifty 500 return over the same period. EXCESS_RET: Portfolio return minus benchmark. The backtest uses a momentum-only screen (RS > 0); fundamental filters are not applied historically due to data limits.

Stock narratives
