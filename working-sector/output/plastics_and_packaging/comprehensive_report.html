How to use this report

This report has six tabs: Overview (sector definition, hypothesis, literature), How to read (this guide), Universe (all stocks with metrics), Shortlist (top names by composite score), Backtest (historical momentum screen results), and Stock narratives (per-stock commentary with fundamental breakdown). Use the tabs to switch views.

Tables: In Universe, Shortlist, and Backtest, click any column header to sort by that column; click again to toggle ascending (▲) / descending (▼). Green-highlighted cells mark the top 25% of values in that column (e.g. highest composite score, best 6M return, highest excess return in backtest).

Price and return metrics

MetricDescription
CURRENT_PRICELatest closing price (Rs).

MetricDescription

Fundamental scores

All fundamental scores are on a 0–100 scale (higher = better). They come from the Screener/organized pipeline.

ScoreDescription
FUND_SCOREOverall fundamental score (earnings quality, sales growth, financial strength, institutional backing combined).
Earnings qualityQuality and sustainability of earnings (margins, consistency, cash flow).
Sales growthRevenue growth trajectory.
Financial strengthBalance sheet health, leverage, liquidity.
Institutional backingDegree of institutional (DII/FII) ownership and trend.

Composite and screen

TermDescription
COMPOSITE_SCOREWeighted combination: 0.4 × fundamental + 0.4 × technical + 0.2 × relative strength rank (0–100). Used to rank and shortlist stocks.
PASS_FUNDTrue if fundamental score >= 70 (quality screen).
PASS_RSTrue if 6M relative strength vs Nifty 500 > 0 (momentum screen).
PASS_SCREENTrue if both PASS_FUND and PASS_RS.

Backtest

REBAL_DATE: Date when the portfolio was formed. N_STOCKS: Number of stocks passing the screen. PORTFOLIO_RET_1Y: Equal-weight return of those stocks over the next year. BENCH_RET_1Y: Nifty 500 return over the same period. EXCESS_RET: Portfolio return minus benchmark. The backtest uses a momentum-only screen (RS > 0); fundamental filters are not applied historically due to data limits.

Stock narratives

Each stock has a short narrative that synthesizes key metrics (price, returns, RSI, technical and fundamental scores) and key financial ratios/fundamentals: P&L (Sales, Net Profit, EPS, YoY), Quarterly (last 4Q sales and profit), Balance sheet (Equity, Debt, Cash, Debt/Equity), and Ratios (ROCE, ROE, EPS, PE, PB, OPM, NPM, Interest Coverage, Div Yield when available). The fundamental score (0–100) is the main financial-quality metric. Run fetch_screener_fundamental_details.R to populate these from Screener.in.

Recommendation: The LLM (Ollama) provides a view-based recommendation (Buy / Accumulate / Hold / Reduce / Avoid) with a one-line rationale derived from the metrics. This is for research context only, not investment advice.
